## Surgishop Reports

Custom ERPNext reports for Surgishop sales tracking and performance management.

### Features

- Regional Dashboard - Sales performance by territory
- Monthly Pacing - Quarterly goal tracking by region
- Director Dashboard - Executive summary of quarterly performance
- Outbound Shipping - Delivery note tracking
- Item Tracking Report - Item inventory and history
- Stock Status - Current inventory levels
- Plus 14+ additional custom reports

### Installation

1. Add this app to your Frappe/ERPNext site
2. Run migrations
3. Access reports from the Reports module
